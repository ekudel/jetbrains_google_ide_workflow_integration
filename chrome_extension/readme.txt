To install the extension to your browser:
1. Open "chrome://extensions" page
2. Enable "Developer Mode"
3. Install unpacked extension using "extension" directory in this module

This extension provides following features:
1. On the page of youtrack issue (if its subsystem is 'Android'), beyond the right-side panel
   "Related AOSP issues: ..." element appears. It contains links to all issues in AOSP tracker mentioned in the page
   and also "New" link that should be used for creating new related issue in AOSP tracker.
